WebSocket соединение для чата

Устанавливает WebSocket соединение для реального времени общения с чат-ботом.

Протокол WebSocket:

Отправка сообщения:

{
  "auth_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "content": "Привет, как дела?"
}

Успешный ответ бота:

{
  "status": "success",
  "content": "Привет! У меня всё отлично. Как дела у тебя?",
  "created_at": "2024-01-15T10:30:45Z"
}

Ответ с ошибкой:

{
  "status": "error",
  "error": "Некорректный токен авторизации",
  "created_at": "2024-01-15T10:30:45Z"
}
Authorizations:
BearerAuth

Responses

Production server

https://tatarstan-card.ru/api/chat/ws
